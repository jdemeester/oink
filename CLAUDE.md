# Project Instructions for Claude Code

## Component Library
This project uses **HeroUI Native** for all UI components.

**Always fetch documentation from:** https://v3.heroui.com/native/llms-components.txt

## Rules
1. Always use HeroUI Native components - never create custom UI when a HeroUI component exists
2. Before implementing UI, reference the llms-components.txt documentation
3. Use HeroUI color tokens: bg-accent, bg-warning, bg-success, bg-danger
4. Look at existing components in src/components for usage patterns

## Available Components (from 'heroui-native')
Button, Card, Avatar, Chip, Surface, Tabs, Accordion, TextField, FormField, Checkbox, Switch, RadioGroup, Select, Dialog, BottomSheet, Popover, Toast, Spinner, Skeleton

## Rebase App
Located in [src/app/(rebase)/](src/app/(rebase)/). Design should be calm, minimal, mobile-first (~390px width).
